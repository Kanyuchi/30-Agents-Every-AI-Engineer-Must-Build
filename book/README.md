# Book

This folder holds a local copy of the book this repository accompanies:

**30 Agents Every AI Engineer Must Build** (ISBN 9781806109012)

- File: `30-Agents-Every-AI-Engineer-Must-Build_9781806109012.pdf`
- The PDF is **git-ignored** (`book/*.pdf`) — it is a copyrighted commercial
  work and is ~84 MB, so it is kept local rather than committed.

Each `chapterNN/` directory in the repo root contains the code for the
corresponding chapter of this book.
